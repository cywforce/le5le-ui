布局 - flex

css3的flex布局比传统grid更方便友好。缺点是老浏览器的兼容问题。

flex本身，是非常强大的。目前，仅仅根据本项目要求，只做了很简单的基础封装（后续有需求在完善）。更多可以参考阮一峰的 Flex 布局教程 。

使用

1. 自适应宽度从左到右排列

使用class: flex进行基本的flex布局。
宽度100px
自生长宽度
宽度20%
填满剩余

2. 自动换行

flex默认是不换行的。当使用 flex wrap组合时，才换行。
宽度100px
自生长宽度
宽度20%
宽度100%

3. 自动填充宽度

当左右两列固定， 中间自动填充时，可以给中间栏加上full类。
宽度100px
自生长宽度
宽度200px
