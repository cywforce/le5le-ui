le5le - UI

le5le - UI是用在le5le各项目中基于html5和css3的css框架。适用于支持html5和css3的浏览器。 主要采用flex、calc等css3新特性去实现布局，更加简单、方便、优雅的实现垂直居中，grid布局等。

在项目构建方面，通过postcss去实现按需导入。采用precss插件，支持scss和less语法。本工程采用的是webpack打包。

安装

1. 安装node和npm。
   npm安装postcss插件： autoprefixer、postcss、postcss-import、precss。如果您使用webpack打包工具，参考本项目中 package.json安装打包所依赖的插件。

2. 编译
   A. webpack按需集成到项目中
    在您的项目中定义一个index.css文件，@import相关模块编译打包即可。可以参考本项目 源码 中的demo。
   B. 直接使用整个css
    下载源码到本地。在命令行执行npm install安装依赖文件，然后在运行编译命令npm run build，在www/assets下的css文件 即为编译后文件，直接在html中引用即可。
